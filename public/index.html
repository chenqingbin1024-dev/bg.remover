常见问题

数据是否安全？

所有图片通过 HTTPS 加密传输，仅用于 remove.bg 实时处理，不会被存储。

如何获取 API Key？

至 remove.bg 控制台申请开发者 Key，并在运行服务前设置 REMOVE_BG_API_KEY 环境变量。

是否支持批处理？

当前版本支持单张排队，后续可扩展为批量上传和队列处理能力。
